Comandos Completos do CSS 3 com Significado

Seletores:

Seletores básicos:

Tipo de elemento: Seleciona todos os elementos do tipo especificado (ex: h1, p, div).

  • #id: Seleciona o elemento com o ID especificado.
  • .classe: Seleciona todos os elementos com a classe especificada.
  • elemento > filho: Seleciona os elementos filhos diretosdo elemento especificado.
  • elemento + irmão: Seleciona o irmão adjacente direto do elemento especificado.

Seletores combinados:

    descendente: Seleciona todos os descendentes do elemento especificado.
  • pai > filho: Seleciona os filhos diretos do elemento pai especificado.
  • elemento ~ irmão: Seleciona todos os irmãos subsequentes do elemento especificado.

Pseudo-classes:

  • :hover: Seleciona o elemento quando o cursor está sobre ele.
  • :focus: Seleciona o elemento quando ele tem o foco.
  • :active: Seleciona o elemento quando ele está sendo ativado (clicado, pressionado).
  • :first-child: Seleciona o primeiro filho do elemento especificado.
  • :last-child: Seleciona o último filho do elemento especificado.

Pseudo-elementos:

  • ::before: Insere conteúdo antes do elemento especificado.
  • ::after: Insere conteúdo depois do elemento especificado.

Propriedades de tipografia:

  • font-family: Define a família da fonte do texto.
  • font-size: Define o tamanho da fonte do texto.
  • font-weight: Define o peso da fonte do texto (normal, negrito, etc.).
  • font-style: Define o estilo da fonte do texto (normal, itálico, etc.).
  • text-align: Define o alinhamento do texto (esquerda, centralizada, direita).
  • text-decoration: Define a decoração do texto (sublinhado, itálico, etc.).
  • text-transform: Define a transformação do texto (maiúsculas, minúsculas, etc.).
  • letter-spacing: Define o espaçamento entre as letras do texto.
  • line-height: Define a altura da linha do texto.

Propriedades de cores:

  • color: Define a cor do texto.
  • background-color: Define a cor do fundo do elemento.
  • background-image: Define a imagem de fundo do elemento.
  • background-repeat: Define como a imagem de fundo se repete.
  • background-position: Define a posição da imagem de fundo.
  • background-size: Define o tamanho da imagem de fundo.
  • border-color: Define a cor da borda do elemento.
  • border-style: Define o estilo da borda do elemento (sólida, tracejada, etc.).
  • border-width: Define a largura da borda do elemento.

Propriedades de layout:

  • width: Define a largura do elemento.
  • height: Define a altura do elemento.
  • margin: Define as margens do elemento (superior, direita, inferior, esquerda).
  • padding: Define o preenchimento do elemento (superior, direita, inferior, esquerda).
  • display: Define o tipo de caixa do elemento (bloco, inline, etc.).
  • float: Define se o elemento flutua para a esquerda ou direita.
  • position: Define o posicionamento do elemento (estático, relativo, absoluto, fixo).
  • clear: Define se o elemento limpa os elementos flutuantes ao seu redor.
  • overflow: Define como o conteúdo do elemento se comporta quando excede seu tamanho.

Propriedades de caixa:

  • box-sizing: Define como o tamanho da caixa do elemento é calculado.
  • box-shadow: Define uma sombra para o elemento.
  • border-radius: Define o raio da borda do elemento.

Propriedades de flexbox:

  • display: flex: Define o layout do elemento como flexbox.
  • flex-direction: Define a direção do layout flexbox (horizontal, vertical).
  • justify-content: Define como os elementos flexbox são distribuídos horizontalmente.
  • align-items: Define como os elementos flexbox são distribuídos verticalmente.
  • flex-wrap: Define se os elementos flexbox podem ser quebrados em várias linhas.

Propriedades de grid:

As propriedades de grid definem o layout de grade para um container, permitindo o posicionamento preciso de elementos filhos. Veja algumas delas:

  • display: grid: Transforma o elemento em um container de grid.
  • grid-template-columns: Define o número de colunas e seus tamanhos.
    • Exemplos: grid-template-columns: repeat(3, 1fr); (cria 3 colunas com tamanho flexível igual) ou grid-template-columns: 100px 200px 1fr; (define larguras fixas para as duas primeiras colunas e uma flexível para a terceira).
  • grid-template-rows: Define o número de linhas e seus tamanhos. Similar a grid-template-columns.
  • grid-gap: Define o espaçamento entre as linhas e colunas do grid.
  • justify-content: Define como os elementos filhos do grid são distribuídos horizontalmente dentro do container (similar ao flexbox).
  • align-items: Define como os elementos filhos do grid são distribuídos verticalmente dentro do container (similar ao flexbox).

Estas são algumas das principais propriedades de grid. Para posicionar elementos filhos dentro do grid, você pode utilizar propriedades como:

  • grid-column: Define em qual coluna o elemento deve ser posicionado.
  • grid-row: Define em qual linha o elemento deve ser posicionado.
  • grid-column-start/grid-column-end: Define o intervalo de colunas que o elemento irá ocupar.
  • grid-row-start/grid-row-end: Define o intervalo de linhas que o elemento irá ocupar.
